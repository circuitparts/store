# View Order History

<Steps>

### Signup or Login

If you haven't already, sign up for a free Circuit Parts account. [Sign up now](https://circuitparts.in/auth/signup) or [Login](https://circuitparts.in/auth/login) if you already have an account.

### View Order History

On the top right corner of the page, click on the user avatar and select **"Order History"** from the dropdown menu. You will be redirected to the order history page where you can view all your past orders in chronological order with the latest order at the top.

### View Order Details
Click on the **"View Order"** button to view the details of a specific order. 

</Steps>
